Pagination

How does it work?

If you first make an unordered link tag, class it as pagination and inside put li tags with the class page-item for how many page options you want including the previous and next text so if you had 3 pages you would need 5 li tags. Now inside the li tags you need to put the a tag and class it as page-link then make a href and put the link you want the page to go or just make it as # to see the buttons only and outside the a tag put the text previous then 1 2 ect. till the pages you want then put next. Note you have to duplicate the code every time for different text.
